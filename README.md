# Community-Pulse-Odoo-Hackathon

## Setup

### Frontend

`cd frontend`

`bun install`

`bun run dev`

### Backend

`cd backend`

`python -m venv .venv`

`source .venv/bin/activate`

`pip install -r requirements.txt`

`fastapi dev main.py`
